# CoinHive

[CoinHive](https://github.com/cazala/coin-hive) Mine cryptocurrency Monero (XMR) using CoinHive from node.js

### 安装

```
npm install -g coin-hive

```
或

```
npm install coin-hive --unsafe-perm=true --allow-root

```
